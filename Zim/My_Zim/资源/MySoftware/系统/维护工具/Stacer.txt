====== Stacer ======
Created Wednesday 24 October 2018

Stacer[43] 是一款用于监控和优化 Linux 系统的免费开源应用程序。

使用以下命令在 Ubuntu 和 Debian 安装 Stacer。

$ sudo add-apt-repository ppa:oguzhaninan/stacer
$ sudo apt-get update
$ sudo apt-get install stacer
