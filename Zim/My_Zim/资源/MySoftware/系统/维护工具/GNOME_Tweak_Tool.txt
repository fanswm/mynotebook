====== GNOME Tweak Tool ======
Created Wednesday 24 October 2018

GNOME Tweak Tool[42] 是用于自定义和调整 GNOME 3 和 GNOME Shell 设置的流行工具。

使用以下命令在 Ubuntu 和 Debian 安装 GNOME Tweak Tool。

$ sudo apt install gnome-tweak-tool
https://github.com/nzjrs/gnome-tweak-tool
