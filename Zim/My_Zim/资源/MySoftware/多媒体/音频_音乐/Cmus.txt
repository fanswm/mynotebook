====== Cmus ======
Created Wednesday 24 October 2018

Cmus[62] 可以说是最高效的的命令行界面音乐播放器了，它具有快速可靠的特点，也支持使用扩展。

使用以下命令在 Ubuntu 和 Debian 安装 Cmus。

$ sudo add-apt-repository ppa:jmuc/cmus
$ sudo apt-get update
$ sudo apt-get install cmus
