====== Ardour ======
Created Wednesday 24 October 2018

Ardour
Ardour[50] 是一款漂亮的的数字音频工作站Digital Audio Workstation，可以完成专业的录制、编辑和混音工作。

使用以下命令在 Ubuntu 和 Debian 安装 Ardour。

$ sudo add-apt-repository ppa:dobey/audiotools
$ sudo apt-get update
$ sudo apt-get install ardour
https://ardour.org/
