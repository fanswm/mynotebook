====== gimp ======
Created Tuesday 21 November 2017

GIMP[52] 是 Photoshop 的开源替代品中最受欢迎的。这是因为它有多种可自定义的选项、第三方插件以及活跃的用户社区。

使用以下命令在 Ubuntu 和 Debian 安装 Gimp。

$ sudo add-apt-repository ppa:otto-kesselgulasch/gimp
$ sudo apt update
$ sudo apt install gimp


https://www.gimp.org/
