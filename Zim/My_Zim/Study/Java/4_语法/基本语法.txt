====== 基本语法 ======
Created Tuesday 27 February 2018


__基本语法__

编写Java程序时，应注意以下几点：

	大小写敏感：Java是大小写敏感的，这就意味着标识符Hello与hello是不同的。
	__类名__：对于所有的类来说，类名的首字母应该大写。如果类名由若干单词组成，那么__每个单词的首字母应该大写__，例如 MyFirstJavaClass 。
	__方法名__：__所有的方法名都应该以小写字母开头__。如果方法名含有若干单词，则后面的每个单词首字母大写。
	__源文件名__：__源文件名必须和类名相同__。当保存文件的时候，你应该使用类名作为文件名保存（切记Java是大小写敏感的），文件名的后缀为.java。（如果文件名和类名不相同则会导致编译错误）。
	主方法入口：所有的Java 程序由__public static void main(String []args)__方法开始执行。
