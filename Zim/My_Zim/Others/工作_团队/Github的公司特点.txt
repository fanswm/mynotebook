====== Github的公司特点 ======
Created Tuesday 05 December 2017

  ● 专注创作，高创意
  ● 运营良好与较高的内外满意度
  ● 高利润，较低的融资额或零融资

  ● 每一位GitHub公司的新员工，官方博客将发表文章欢迎。
  ● 在GitHub内部，没有经理，需求内容与优先级由项目组自行决策。
  ● 选择自己的工作时间、工作地点。
  ● 员工来自开源社区。
  ● 能开源的尽可能开源。
