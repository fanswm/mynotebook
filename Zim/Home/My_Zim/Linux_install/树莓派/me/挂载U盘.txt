====== 挂载U盘 ======
创建日期 星期三 21 十一月 2018

挂载U盘：
$ sudo mkdir /mnt/usb
$ sudo vi /etc/fstab
#添加
/dev/sda    /mnt/myusbdrive    auto    defaults,noexec,umask=0000    0    0
$ sudo mount -o iocharset=utf8,uid=1000,gid=1000 /dev/sda1 /mnt/usb/
