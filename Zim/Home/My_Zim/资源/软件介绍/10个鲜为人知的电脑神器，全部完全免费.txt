====== 10个鲜为人知的电脑神器，全部完全免费 ======
Created 星期一 18 一月 2021

https://zhuanlan.zhihu.com/p/313032213

1：Lively Wallpaper
Lively Wallpaper 是一个免费开源的电脑桌面动态壁纸软件；软件仅支持 Windows 版本。

它完全免费，良心好用，内置 12 个精美的动态壁纸，支持将你电脑本地的视频、GIF 动图设置为桌面壁纸，还支持将网页、游戏转换为桌面壁纸。

它的界面简洁清新，打开软件就可以看到这 12 个精美的动态壁纸，鼠标左键单击任意一个壁纸就可以设置为桌面壁纸，你也可以鼠标右键预览壁纸、设置为壁纸、打开文件所在位置。


它的每一个动态壁纸都是非常炫酷，画质超清，交互生动自然。

例如这个动态壁纸，当我们移动鼠标、单击鼠标、按住鼠标左键拖动，都会看到非常酷炫的色彩效果，并且每一个鼠标操作，色彩都会不一样。


这个动态壁纸是一个化学元素周期表，我们点击壁纸上方的元素分类，会突出展示同类型化学元素，点击任意一个化学元素，会放大展示这个化学元素，还会展示这个化学元素的相关信息。


它提供的 12 个动态壁纸，其中绝大部分动态壁纸都支持自定义壁纸功能，包括：颜色、背景、动态壁纸的速度等。

我们看到的这个动态下雨壁纸就支持自定义壁纸功能，包括下雨的强度、速度、亮度，还有雨水的大小，下雨的背景。


它的自定义动态壁纸功能支持滑动调节，你可以实时预览壁纸效果，非常的方便舒适。


最后，除了软件提供的 12 个动态壁纸，你也可以点击软件主界面左上角的加号按钮添加本地视频和 GIF 设置为桌面壁纸。

你还可以在软件的设置中，将软件设置为开机自启动。

总体而言，它完全免费，操作简单便捷，效果出色，支持丰富的自定义功能，是一个值得尝试的电脑桌面动态壁纸软件。


2：HiBit Uninstaller
HiBit Uninstaller 是一个完全免费的电脑专业卸载软件；软件仅支持 Windows 版本。

它完全免费，非常良心，颜值极高，支持可安装版本和便携版本，其中便携版本体积不到 10M ，非常小巧，不需要安装，直接双击就可以打开使用。

第一次打开软件，我们需要点击软件右上方，找到语言选项，将软件默认的英文界面设置为中文。

它支持自动扫描电脑已安装的软件，你可以按照程序名、安装日期、大小、类型排序展示已安装的软件，还可以直接搜索你需要卸载的软件。


右键任意一个需要卸载的软件，可以看到卸载和强制删除选项，如果你遇到普通卸载没有用的情况，可以使用强制删除。

它的卸载功能操作简单，效果非常出色，并且支持卸载之后扫描软件残留的垃圾文件和注册表项，还支持批量卸载功能。

除了卸载功能，它还支持非常多实用的功能，点击软件主界面右上角的工具按钮就可以看到这些功能，包括：浏览器扩展管理器、注册表清理、垃圾文件清理、空文件夹清理、快捷方式修复、进程管理器、启动管理器、计划程序任务管理器等。


3：Captura 录屏
Captura 录屏是一个完全免费并且非常良心的电脑录屏软件；软件仅支持 Windows 版本。

它是一个免费软件，也是一个开源软件；它功能强大，界面简洁清新，体积小巧 (只有 300 多 KB ) ，使用非常流畅。

首先，软件打开默认英语界面，我们可以直接在软件的设置里面将软件语言修改为中文，然后软件就会变成中文界面，我们使用起来就会非常轻松。


它主要的功能就是录屏，可以说该有的录屏功能它都有，并且每个功能都是非常全面细致，它还支持很多实用的细节功能。

它支持 6 种录屏功能，包括：只录制音频、全屏录制、录制某一个窗口、录制某一个区域、游戏录制。

每种录制功能都是非常好用全面，其中录制某一个区域支持设置录制区域的宽高，你可以直接拖拽设置，也可以输入数字设置。

它还支持很多实用的细节功能，包括：支持多种视频格式的录制、支持 GIF 格式录制、支持捕获鼠标、支持捕获鼠标点击、支持捕获键盘按键、支持画笔功能、支持截图功能、支持计时功能、支持混合从麦克风和扬声器输出录制的音频。


4：BookxNote Pro
BookxNote Pro 是一个完全免费的宝藏 PDF 笔记软件；软件仅支持 Windows 版本。

首先，它不仅支持打开 PDF 文件，也支持打开 EPUB 文件，主要的功能包括：PDF 阅读、PDF 标注、PDF 记笔记、PDF 笔记整理、PDF 笔记思维导图，可以让你边看 PDF ，边做笔记和思维导图。

它完全免费，非常良心，功能强大，是一个非常好用的电子学习笔记软件，功能和 MarginNote 软件类似，MarginNote 软件只有 macOS 和 iOS 版本。


打开软件，你会看到一个极简的界面，点击左上角的加号按钮就可以导入 PDF 文件或者电子书籍，软件支持自动导入书籍目录功能。

导入完成，我们进入到 PDF 编辑界面，这里支持多种 PDF 显示，包括：双文档并列显示、双文档水平布局、双文档垂直布局、双页连续、适合页面，也支持鼠标滚轮放大缩小页面。

点击鼠标右键，支持前往第一页、上一页、下一页、最末页、指定页，支持对 PDF 左旋转、右旋转、全显示、垂直拆分视图、拆分视图、双页模式单独显示封面。


它支持丰富的 PDF 编辑标注工具，包括：文本、直线、荧光笔、矩形、圆形、套索工具、高亮、下划线、文本框、涂鸦、橡皮檫。

每个工具都支持设置颜色和画笔的粗细。


它支持笔记自动添加到大纲功能，你可以在阅读 PDF 的时候对文档进行重点笔记标注和批注，一边阅读，一边画重点，并且支持随时调整笔记脑图的逻辑结构。

它支持在标注重点的时候，笔记脑图可以窗口并列显示，点击任意一个笔记内容，会自动跳转到对应的文章位置。

点击最左侧的三个笔记按钮，分别是：线性笔记、脑图、大纲，三种笔记形式都支持编辑笔记，而且支持自动同步。

例如你在大纲里面进行了编辑操作，线性笔记也会进行同步变化。


它的大纲也支持进一步编辑功能，包括：节点上移、节点下移、回退节点、缩进节点、删除复制剪切节点内容、颜色设置。

它的脑图功能支持将笔记一键形成思维导图，支持竖向思维导图和横向思维导图，支持用鼠标拖动思维导图修改层级关系，支持双击思维导图节点继续编辑笔记，支持设置思维导图的参数，支持设置思维导图节点的颜色。


它支持笔记分类功能，你可以同时对多本书阅读做笔记，并且支持多种笔记本显示样式，强大的笔记本搜索功能，强大的笔记搜索功能。

最后，它支持三种主题，包括：浅色、暗色、护眼，支持多种格式的笔记导出，包括：大纲 HTML 、线性笔记 HTML 、Anki 格式、思维导图 PNG 文件、Xmind 思维导图格式、OPML 文件。


5：Coodesker
Coodesker 是一个小巧免费的电脑桌面整理软件，特别好用；软件仅支持 Windows 版本。

它不需要安装，双击打开就可以使用，并且体积小巧，只有 5M ，干净舒适，没有任何多余的东西，实际体验非常舒适。

双击软件，软件自动进行桌面整理，将桌面文件和软件图标放入到不同的盒子块中。


你可以右键桌面进入到设置中心将软件设置为开机启动，也可以选择自动整理新增文件和整理快捷方式文件。

它的盒子标签不仅支持竖向显示，也可以设置为横向显示。

除了盒子默认提供的标签，包括：图片、文档、目录、压缩、其它，你也可以新建分类，设置分类的标题和对应的文件后缀。


最后，它还支持创建多个盒子，对每个盒子进行重命名，你可以直接拖拽文件到盒子中分类整理桌面，也可以直接拖动盒子改变盒子的位置。


6：FolderMove
FolderMove 是一个特别实用的 C 盘搬家软件，它可以保证你电脑软件正常使用的同时，给你的 C 盘瘦身，释放 C 盘的容量；软件仅支持 Windows 版本。

它的体积非常小巧，仅 1.25M ，并且不需要安装，直接双击就可以运行。

它的使用方法非常简单，首先选择你需要移动的软件对应在 C 盘的文件夹，这里支持浏览定位文件夹，并且一次只能移动一个软件的文件夹。

然后选择你需要移动到的目标文件夹，这里同样支持浏览文件夹，还支持新建文件夹，最后直接点击移动并设置符号链接按钮即可。


7：EarTrumpet
EarTrumpet 是一个让人眼前一亮的电脑音量辅助工具，主要的功能是可以单独调整每个应用的音量；软件仅支持 Windows 10 。

它支持免费使用，并且使用方法非常简单，打开软件，任务栏的系统音量控制图标旁边会出现一个一摸一样的音量控制图标，直接点击这个图标就可以单独调整每个应用的音量。


鼠标右键这个图标按钮，我们可以看到高级和设置，高级包括：音量合成器、播放设备、录制设备、声音、声音设置，设置支持配置快捷键。


8：火苗
火苗是一个完全免费的电脑批量改名软件；软件仅支持 Windows 版本。

它完全免费，体积小巧，不到 2M ，并且界面极简，颜值非常高；它的功能强大，整个软件实用便捷。

它支持非常实用的改名功能，可以点击选择文件夹导入文件，也可以直接拖动导入文件。


它支持多个改名工具，包括：整体、替换、删除、添加、文本、命令、元变量，其中整体支持的功能包括文件名全部大写、全部小写、首字母大写，也支持扩展名全部大写和小写。

它还支持删除替换文件名中的部分字符，其中添加支持的功能包括：文件名前添加、文件名后添加、从文件名第几个字符添加字符串。

设置完成之后，你可以先不点击应用按钮，可以先点击预览按钮预览改名后的效果。


除了强大的改名功能，它还支持图片批量编辑功能和批量修改文件日期功能，其中图片批量编辑功能支持批量修改图片尺寸。

你可以设置需要修改到的图片尺寸，软件也提供了很多常用尺寸，包括：1 寸照片、大 1 寸照片、2 寸照片等。


9：Pomy
Pomy 是一个完全免费并且颜值非常高的电脑定时提醒休息软件；软件不仅支持 Windows 版本，也支持 macOS 版本。

它的颜值非常高，支持多个外观样式，你可以随意切换。


它的使用方法非常简单，打开软件，点击圆形播放按钮就开始计时，20 分钟之后你的电脑桌面会变黑，这个时候软件就提醒你休息，休息时间是 20 秒。

值得需要注意的是，它不支持自定义间隔时长和休息时长，并且在 20 秒休息的时候你不能进行其它电脑操作，如果你想继续工作，你可以点击 Esc 键退出休息。


10：匍匐前进的布朗熊
匍匐前进的布朗熊是一个非常有趣的电脑动态屏保软件；软件不仅支持 Windows 版本，也支持 macOS 版本。


它的使用方法非常简单，我们只需要安装软件，然后软件会自动弹出屏幕保护程序设置，你只需要设置屏幕保护程序为 LINEScreenSaverInst_Brown ，然后设置等待时间，点击确定应用按钮就可以设置完成。

最后我们可以看到动态屏保效果，一只非常可爱的布朗熊在盯着你的同时自左向右匍匐前进，并且会伴随着布朗熊前进与地面发生摩擦的声音。
