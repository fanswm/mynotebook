====== 我在 GitHub 找到北大清华、浙大上交的计算机课程资源 ======
创建日期 星期一 22 四月 2019

3 月份公号推送了 《我在 GitHub 上找到了中科大的计算机课程资源》。后来应该是因为版权原因，该项目的作者 mbinary 删除了。







后来 GitHub 上又有了另外一个版本的中科大课程资源分享的 repo，「课程关系」图和 mbinary 版的一样，如下：





 

中科大课程资源 项目的地址是：

https://github.com/USTC-Resource/USTC-Course



GitHub 其实除了有中科大的课程资源，还有国内其他一些知名高校的课程资源。



浙江大学课程资源



这个项目已包括了大部分计科专业课程。链接：

https://github.com/QSCTech/zju-icicles
