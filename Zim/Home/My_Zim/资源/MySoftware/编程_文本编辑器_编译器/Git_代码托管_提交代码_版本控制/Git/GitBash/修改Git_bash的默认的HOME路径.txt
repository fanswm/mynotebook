====== 修改Git bash的默认的HOME路径 ======
Created 星期六 31 八月 2019

https://blog.csdn.net/qq_39932010/article/details/78912550

打开/etc/profile.d/env.sh
在“__export PATH="$HOME/bin:$PATH"__ ”之上加入以下三行：
···
HOME="你想修改成为的HOME目录"
HOME="$(cd "$HOME" ; pwd)"
cd
···
export PATH="$HOME/bin:$PATH"

HOME="/d/github/"
HOME="$(cd "$HOME" ; pwd)"
cd
