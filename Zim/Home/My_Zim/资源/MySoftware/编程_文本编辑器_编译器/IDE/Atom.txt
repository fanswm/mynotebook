https://ide.atom.io/
Atom IDE[29] 是一个加强版的 Atom 编辑器，它添加了大量扩展和库以提高性能和增加功能。总之，它是各方面都变得更强大了的 Atom 。

使用以下命令在 Ubuntu 和 Debian 安装 Atom。

$ sudo apt-get install snapd
$ sudo snap install atom --classic
