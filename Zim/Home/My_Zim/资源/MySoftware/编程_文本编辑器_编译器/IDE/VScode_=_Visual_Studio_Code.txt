====== VScode = Visual Studio Code ======
Created Wednesday 24 October 2018

Visual Studio Code[31] 是由微软开发的代码编辑器，它包含了文本编辑器所需要的最先进的功能，包括语法高亮、自动完成、代码调试、性能统计和图表显示等功能。

参考阅读：在Ubuntu 下载 Visual Studio Code[https://code.visualstudio.com/download]

Getting Started： https://code.visualstudio.com/docs?start=true
