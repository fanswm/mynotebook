====== Skype ======
Created Wednesday 24 October 2018

Skype[https://www.skype.com/] 也是一个广为人知的软件了，任何感兴趣的用户都可以在 Linux 上使用。

使用以下命令在 Ubuntu 和 Debian 安装 Skype。

$ sudo apt install snapd
$ sudo snap install skype --classic
