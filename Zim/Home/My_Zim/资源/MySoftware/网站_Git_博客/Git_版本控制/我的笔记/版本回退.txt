====== 版本回退 ======
Created Monday 05 November 2018

+ 查看历史记录：
`git log`

+ 如果嫌输出信息太多，看得眼花缭乱的，可以试试加上--pretty=oneline参数：
`git log --pretty=oneline`

> Git的commit id不是1，2，3……递增的数字，而是一个SHA1计算出来的一个非常大的数字，用十六进制表示
> 在Git中，用HEAD表示当前版本; 上一个版本就是HEAD^，上上一个版本就是HEAD^^


回退到上一个版本
`git reset --hard HEAD^`
