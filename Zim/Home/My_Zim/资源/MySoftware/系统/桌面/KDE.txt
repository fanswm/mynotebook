====== KDE ======
Created Wednesday 24 October 2018

KDE[41] 由 KDE 社区开发，它为用户提供图形解决方案以控制操作系统并执行不同的计算任务。

使用以下命令在 Ubuntu 和 Debian 安装 KDE。

$ sudo apt install tasksel
$ sudo apt update
$ sudo tasksel install kubuntu-desktop
https://www.kde.org/plasma-desktop
