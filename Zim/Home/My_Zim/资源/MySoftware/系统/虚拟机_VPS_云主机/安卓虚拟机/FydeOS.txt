====== FydeOS ======
Created Monday 31 December 2018

FydeOS是一款基于Chrome OS电脑平台端的国产系统，目前刚公测，可以实现在此系统上同时运行网页程序 + 安卓程序 + Windows 程序 + Linux 程序，只需一个U盘即可体验一番。
https://fydeos.com/

好多人都知道基于安卓X86的remix OS之类的可安装到电脑上的安卓系统，但今天小编说的不是它们，小编今天说的厉害了，是基于谷歌Chrome OS的国产操作系统，能实现Windows+安卓+Linux三合一系统。



一、简介



FydeOS是一款基于Chrome OS电脑平台端的国产系统，目前刚公测，可以实现在此系统上同时运行网页程序 + 安卓程序 + Windows 程序 + Linux 程序，只需一个U盘即可体验一番。

三、U盘系统制作



下面开始说说使用教程，其实官网写的也很详细，小编就从头到尾说说。

1、下载系统镜像：

如下官网下载通用的镜像即可，格式为xz，大小1.51G。






2、U盘启动工具制作

目前公测版只能用U盘方式安装，后续可能会加入虚拟机或者硬盘版。

我们准备一个8G以上U盘（建议USB 3.0的，读写速度快），需要格式化。如下采用官网推荐的U盘启动工具etcher制作即可，步骤很简单：插入U盘—选择xz镜像文件—点击flash即可。

注意2：制作完成看到我的电脑，一堆分区，这是正常的，请忽略。








四、系统实测



上一步完成U盘启动制作，不用拔U盘。我们重启电脑，选择U盘启动（U盘启动方式，自行百度BIOS设置U盘启动，此处不赘述），一切默认，即可进入FydeOS的世界。

1、主界面

主界面就是桌面系统的模样，维持了谷歌一贯的扁平化MD风格。操作流畅，舒服的过渡动画，自带谷歌浏览器，你刚需的截屏键：ctrl+shift+F5。






2、安卓系统在哪？

根据官方描述，系统运行后，会自动下载安卓支持插件，这个时间小编没测试。点击左下角（类似于Windows的开始键），如下有安卓选项代表安装成功，可以看到是基于安卓7.1.1的。






然后系统就可以安装安卓软件啦，浏览器下载即可，如下小编安装了TIM和酷安。






3、Windows系统在哪？

Windows的支持环境小编当然也给大家准备好了，叫做CrossOver Chrome OS，也是个安卓安装包，大家直接安装即可。

然后如下选择exe安装包即可，比如小编测试WPS 2019电脑版，可成功安装运行。






4、Linux系统在哪？

Linux属于小编盲区，也是在系统设置里可以找到，此处不会写了。





由于目前刚公测，所以bug肯定会有，但经过小编的体验还是相当不错的，毕竟基于谷歌嘛，三平台支持也是很有意思，大家可以试试吧，仅需一个U盘。
