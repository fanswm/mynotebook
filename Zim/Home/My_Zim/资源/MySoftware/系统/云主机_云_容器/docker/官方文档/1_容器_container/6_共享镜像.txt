====== 6 共享镜像 ======
创建日期 星期二 05 三月 2019

__标记镜像：__
docker tag 你的镜像名称  你的dockerhub用户名/你起的仓库名称:tag

__发布镜像__
将已标记的镜像上传到镜像仓库：
docker push  你的dockerhub用户名/你起的仓库名称:你标记的tag

从远程镜像仓库中拉取并运行镜像
从现在开始，您可以使用 docker run，并且可以使用以下命令在任何机器上运行您的应用：
docker run -p 4000:80 username/repository:tag
