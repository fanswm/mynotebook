====== 删除 ======
创建日期 星期二 12 三月 2019

docker rmi <imagename>            # 从此机器中删除指定的镜像
docker rmi $(docker images -q)             # 从此机器中删除所有镜像
先删容器，再删镜像。
如果有两个或多个镜像的IMAGEID相同，则：
sudo docker rmi dockerhub-username/image-name:tag1 dockerhub-username/image-name:tag2
