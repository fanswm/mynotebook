====== vim用法 ======
Created Friday 24 November 2017

## 删除
__+ 所有空行:__
  + `：g/^\s*$/d`
  + `%s/^\n//g`
__+ 行首空格:__
  + `:%s/^ *//g` 
__+ 行首空白:__
  + `:%s/^\s*//`
__+ 行尾的空格:__
  + `:%s/ *$//g` 
  
## 添加
__+ 在行首加一个#号:__
  + `：0，$s/^/#/gc`
__+ 在6~10行的行首加一个#号__
  + `:6,10s/^/#/gc`


__vim删除所有"^M"：__
:%s/^M$//g
（注意其中的“^M”不是直接输入的，必须 按Ctrl+V,再回车）
vim删除所有行首空白：
：%s/^\s*//
