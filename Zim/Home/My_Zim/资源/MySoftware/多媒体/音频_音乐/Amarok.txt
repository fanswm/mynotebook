====== Amarok ======
Created Wednesday 24 October 2018

Amarok[60] 是一款功能强大的音乐播放器，它有一个直观的 UI 和大量的高级功能，而且允许用户根据自己的偏好去发现新音乐。

使用以下命令在 Ubuntu 和 Debian 安装 Amarok。

$ sudo apt-get update
$ sudo apt-get install amarok

https://amarok.kde.org/en
