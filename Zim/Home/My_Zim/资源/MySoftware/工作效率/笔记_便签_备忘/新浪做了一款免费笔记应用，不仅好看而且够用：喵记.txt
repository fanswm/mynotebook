====== 新浪做了一款免费笔记应用，不仅好看而且够用：喵记 ======
创建日期 星期三 29 五月 2019

https://sspai.com/post/54906
喵记是由新浪邮箱团队设计和开发的一款笔记应用，它的界面简洁又功能丰富，集成了笔记、待办事项和提醒功能于一身，还支持 Markdown 语法、搜索和长图分享等功能。如果你在寻找一款轻量级的笔记应用，那么喵记是一个不错的选择。

好看的卡片式设计，单手操作没问题
提到笔记应用，我相信你的脑海里最先浮现的就是 Evernote、Bear、Ulysses 之类的 App，以及它们的经典结构——扁平化列表式布局。

喵记采用了完全不同的设计：每条笔记都是一张小卡片，阴影效果营造的层次感让每张小卡片看起来都像悬浮在屏幕上。喵记的笔记卡片也直观显示了笔记的标题和头部文字，但大字号搭配卡片式设计确实非常好看。


左：Evernote，右：喵记
喵记的界面只有一屏，创建笔记、切换笔记本以及筛选功能位于屏幕下方，使用单手操作完全没有问题。所有的笔记卡片以创建时间顺序排列在应用页面中，随着不断使用，笔记数量增多，使用搜索、按条件筛选以及切换笔记本，都能快速地找到特定笔记。

有趣的是，切换笔记本按钮的图标是一只可爱的喵，它会时不时地眨眼睛或者瞪瞪眼，这个贴心的小细节给整个应用增添许多趣味。


喵记的主界面和功能
够用的记录功能，支持多种分享方式是亮点
我们可以在喵记里轻松地记录一篇笔记，插入照片，为它设置提醒，或者添加待办事项。最值得称赞的莫过于喵记支持 Markdown 语法，并且支持通过标识符和快捷图标设置文本格式。

在喵记的首页，我们可以直接长按笔记来对笔记进行编辑。不得不说喵记的动画效果流畅又精致，在应用内部的几乎每次点击都有精心设计的动画过渡，搭配卡片化简约设计，整个应用的体验让人很舒服。


喵记的动画效果
在每条笔记卡片的底部，显示了我们可以对笔记内容进行的操作：

添加图片：图片会以 Markdown 代码语言形式显示；
添加待办事项：应用会统计笔记中待办事项的完成度；
设置提醒：可以设置按照每 n 分钟提醒，直到点开才结束；
设置颜色：可以为重要程度不同的笔记卡片设置彩色背景，方便一眼就能看到；
Markdown：包括标题、加粗、列点、链接等几类常用语法。

支持 Markdown，能设提醒和颜色
喵记的另一个亮点是可以把笔记制作成精美的图片进行分享，支持生成长图和 PDF，也可以拷贝成文本。所见即所得，喵记生成的图片和在编辑时的内容基本保持一致，比如 Markdown 的渲染样式、底色，还有图文混排。凭借这样的颜值，喵记笔记发到社交媒体，完全不输其它便签应用。


使用喵记分享笔记
小结
喵记主打「随心记录，简单可靠」，确实，它可以满足最基本的笔记需求，同时把这些功能体验做得好用好看。如果你需要一个好看的笔记应用，记录一些简单的旅行、日记，或者是待办清单，不妨试一试这款喵记。

你可以在 App Store 免费下载喵记，应用目前没有适配 iPad 版本。
