====== byte ======
Created Monday 05 March 2018

 __byte 数据类型是8位、有符号的，以二进制补码表示的整数；__
最小值是 -128（-2^7）；
最大值是 127（2^7-1）；
默认值是 0；
byte 类型用在大型数组中节约空间，主要代替整数，因为 byte 变量占用的空间只有 int 类型的四分之一；
例子：byte a = 100，byte b = -50。
