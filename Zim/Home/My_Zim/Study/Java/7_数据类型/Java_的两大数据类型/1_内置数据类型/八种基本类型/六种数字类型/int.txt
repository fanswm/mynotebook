====== int ======
Created Monday 05 March 2018

 __int 数据类型是32位、有符号的以二进制补码表示的整数；__
最小值是 -2,147,483,648（-2^31）；
最大值是 2,147,483,647（2^31 - 1）；
一般地整型变量默认为 int 类型；
默认值是 0 ；
例子：int a = 100000, int b = -200000。
