====== 2 可变参数 ======
Created Sunday 28 January 2018

把函数的参数改为可变参数：

```
def c__alc(*numbers)__:
	sum = 0
	for n in numbers:
		sum = sum + n * n
	return sum
```

__如果传入的参数是list 或 tuple:__
__ calc(*nums)__


可变参数允许你传入0个或任意个参数，这些可变参数在函数调用时自动组装为一个tuple。

__*args是可变参数，args接收的是一个tuple；__
