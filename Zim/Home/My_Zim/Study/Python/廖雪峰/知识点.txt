====== 知识点 ======
Created Thursday 11 January 2018

+ __整数运算__永远是精确的（除法难道也是精确的？是的！），而浮点数运算则可能会有四舍五入的误差。

+ __不可变对象__
str是不变对象，而list是可变对象。

+ __地板除:__
	还有一种除法是//，称为地板除，两个整数的除法仍然是整数：

>>> 10 // 3
3

+ 1行代码能实现的功能，决不写5行代码。请始终牢记，代码越少，开发效率越高。
