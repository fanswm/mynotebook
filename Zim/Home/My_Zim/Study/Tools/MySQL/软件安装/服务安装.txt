====== 服务安装 ======
Created Friday 07 December 2018

运行cmd，将路径切换到：C:/Program files/MySQL server  6.0/bin下，输入命令mysqld-nt --install，如果安装成功会有提示“Service successfully installed”

接下来打开：“控制面板->管理工具->服务”，你就会发现其中多一项服务MySQL，以后您就可以通过它来启动MySQL了。当然当服务安装好后，您还可以通过命令行net start mysql来启动MySQL

另外停止方式也有对应的几种：1、net stop mysql（这样关闭的前题是安装了服务）；2、mysqladmin -u root shutdown（在设置了密码的情况下的命令为mysqladmin -u root -p shutdown）；
3、通过Windows的服务程序进行启动(这样是最直观的）。
