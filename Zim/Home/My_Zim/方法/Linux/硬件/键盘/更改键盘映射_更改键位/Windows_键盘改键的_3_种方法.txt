====== Windows 键盘改键的 3 种方法 ======
Created 星期一 19 八月 2019

https://sspai.com/post/56190
最近我的朋友买了一个蓝牙机械键盘，想同时在 Mac 和 Windows 电脑上切换使用。但是键盘的键帽无法选择，只有适合 Windows 的键盘布局。我们都知道，Windows 和 Mac 键盘最明显的区别之一就是左下角的 Windows 键 和 Alt 键的位置不同，习惯了 macOS 键位的朋友会在使用快捷键时比较别扭，此时就需要对按键进行互换，让它们更符合使用习惯。

还有一些键盘会提供丰富的自定义快捷键，但如果手上只有老式的键盘该怎么办呢？没关系，AutoHotkey 也能帮你将闲置的菜单键等按键变成功能丰富的 Hyper 键。所以本文就来讲一讲在 Windows 中为键盘改键的 3 种方法。

注：macOS 用户可以移步 Power+ 1.0 中的《macOS 改键利器：Karabiner-Elements 使用详解》 一文，查看如何通过 Karabiner-Elements 来进行改键。

用 MapKeyboard 互换按键位置
将两个按键交换的情况，最常见的有两种情况：

键盘布局不同，比如 macOS 用户使用 Windows 时，可能会希望让 Windows 键和 Alt 键进行互换，保持外接键盘和 Mac 键盘布局保持一致。

个人按键习惯不同，比如让 ESC 键和 CapsLock 键、让 FN 键和 Ctrl 键进行互换来改善手感和提高效率。

这种情况是比较容易调整的，我们只需要安装一款名为 MapKeyboard 的软件。这款工具提供了可视化的键位改变方法，不需要手动编写配置文件，非常易于使用。


MapKeyboard
目前 MapKeyboard 的最后一个免费版本是 2.1 版，如今开发者已经将它作为一项功能加入了他新的收费软件 Hotkeycontrol 中。你可以在这里下载到 MapKeyboard 最后的免费版。
