====== MySQL ======
Created Friday 01 December 2017

my mysql
免安装版：
下载mysql-5.6.11-win32.rar
解压到D:\mysql56;
添加环境变量：
path: D:\mysql56\bin
 
以下未验证是否需要：
mysql_home: d:\mysql56\
mysql_lib:d:\mysql56\lib
编辑：D:\mysql\my.ini
basedir="D:\mysql56"
datadir="D:\mysql56\data"
 
cmd.exe
$ cd d:\mysql56\bin
$ mysqld --install
 
启动服务：
$ net start mysql
 
停止服务：
$ net stop mysql
 
第一次登录：
$ mysql -u root -p
无密码；
