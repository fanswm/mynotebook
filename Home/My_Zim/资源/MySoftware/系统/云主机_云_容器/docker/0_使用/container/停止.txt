====== 停止 ======
创建日期 星期二 12 三月 2019

docker stop <hash>                     # 平稳地停止指定的容器
	`docker stop container-name/container-id`
docker kill <hash>                   # 强制关闭指定的容器

容器stop（或Ctrl+D）时，会在保存当前容器的状态之后退出，下次start时保有上次关闭时更改。而且每次进入attach进去的界面是一样的，与第一次run启动或commit提交的时刻相同。
