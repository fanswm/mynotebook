====== 没有 Google 服务不要紧，「微软全家桶」内也有别样的 Android 生态 ======
创建日期 星期三 29 五月 2019

https://sspai.com/post/54886

在网络上，我们时常会听到这样一种论调：「没有 Google 的 Android 只能称作安卓，真正的 Android 体验是不输 iOS 的」。然而，Google 服务虽好，国内却无法正常使用，与之相对的，Windows 用户基数却十分庞大，因此在移动操作系统折戟过的微软也渐渐为 Android 量身打造出了一个由微软应用构建的移动生态。

如果你想体验一种别样的 Android 生态，不妨也试试装一套「微软全家桶」，配合刚推送的 Windows 10 五月更新，打造手机和电脑协同互通的无缝体验。

从启动器开始
启动器是 Android 设备的门面，也是我们最常见到的界面之一，而微软桌面就是一款颇有特色的启动器。它采用了与 Windows 10 类似的设计语言，在负一屏提供了资讯、日程、待办、便签、历史活动记录（时间线）等信息，还支持添加第三方小部件，实现与 iOS 通知中心小组件类似的体验。

微软桌面负一屏
微软桌面负一屏
登录你的微软账户后，便签、待办和时间线都会被同步至你的其它 Windows 10 设备，此处记录，他处可及。不过，由于微软便笺则没有网页版，也不能添加图片，无法完全取代 Google Keep，但 Microsoft To-Do 的完成度相当高，可与 Todoist 一战。


与 Windows 10 双向同步的便笺和提醒

当然，启动器该修的基本功，微软桌面也不会少。你可以用图标包彰显个性、通过手势操作提高效率、切换不同的搜索引擎，并备份和还原设置，喜欢哪种姿势都没问题，加上智能的负一屏，与 Nova Launcher 相比也毫不逊色。

与所有设备保持同步
如果你还想跨平台同步自己的通讯录、日程安排、电子邮件和任务等的话，不需要科学手段的 Microsoft Exchange 服务也是个不错的选择。只需在系统设置或日历、邮件等应用中登录自己的微软账户，即可轻松同步自己的各种信息，不用求助 XX 同步助手了。

在日常生活中，我也使用 Outlook 邮箱作为主账户，将自己的课程表、联系人、日程活动等都交给它处理，从而在任意设备上都能及时同步更改，纵享丝滑。Office 365 赠送的 Outlook 高级版也带来了无广告和 50 GB 邮箱空间等特性，可以说是 Gmail 的最佳替代品。

全平台的 Edge

作为 Internet Explorer 的继任者，Edge 被微软寄予了厚望，却还是难逃成为 Chrome 下载器的命运。或许是意识到了这一点，微软近期宣布将 Edge 内核更换为 Chromium，兼容 Chrome 扩展，并相继放出了 Windows 和 macOS 测试版。至此，Edge 真正成为了可与 Chrome、Firefox 一较高低的全平台浏览器。

Microsoft Edge macOS 版
Microsoft Edge macOS 版
以 Android 版 Edge 为例，你不仅可以与其它设备同步收藏夹、浏览历史和已保存的密码等，还能一键将正在阅读的标签页发送至其它设备，无缝切换。而翻译页面、内容拦截器和新闻分级等特色功能也使其成为了 Chrome 的有力竞争对手，在国内就更是如此了。

同步照片与文档
尽管 Google 相册提供了高画质照片的无限容量上传，但在国内网络环境下，想使用这一功能并不那么容易。而订阅 Office 365 就有 1 TB 云端空间的 OneDrive 在各方面都是个实惠的选择，只是直连速度可能不如国内网盘。

只需要安装 OneDrive 客户端，打开相机上传选项，它就会自动在后台同步你的照片和视频至云端。对于已上传的照片，OneDrive 提供了自动标签分类和生成相册等功能，但个人使用体验不如 Google 相册准确，还待进一步改进。

相机上传/自动相册/智能标记
相机上传/自动相册/智能标记
除了照片外，Office 365 用户还可以在手机上浏览并编辑 OneDrive 中的文件，随时随地都能办公。Office 三剑客的首页会直接列出最近使用过的文档，并支持自动下载缺失的字体、调整排版以适应手机布局等功能，比 Google Docs 体验更佳。

你好，小娜
从被安置在 Windows 10 任务栏最明显的位置就能看出，微软想让 Cortana 成为你真正的个人数字助理。尽管跻身于 Android 下，但 Cortana 也逐渐发展出了自己的特色，例如与微软桌面深度集成，支持连接 Outlook、Office 365、Gmail 等服务，与 Windows 同步提醒内容，提供个性化的今日一览等。

Cortana
Cortana
不过，经过一段时间的体验，我必须要说，Cortana 在 Android 上依然近乎于一个「人工智障」。它不仅网络连接会时而出错，对「明天天气怎么样」这种简单的问题都只会调出搜索。与之形成对比的是，Siri、Google 助理、三星的 Bixby 和小米的小爱同学都会直接给出答案，并语音播报，体验更佳。

结语
在国内的大环境下，享受原汁原味的 Google 产品体验并非易事，而微软全家桶就是在此之外的佳选。如果你正在使用 Windows 设备或者无法克服网络条件限制，覆盖范围全面、综合质量拔尖的微软系 App 也许能成为你的新欢。
