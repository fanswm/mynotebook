====== 无限使用heap内存 ======
Created Thursday 07 December 2017

错误的写法：

byte[] pdf = toPdf(file);

这里有一个前提，就是文件大小不能讲JVM的heap撑爆。否则就等着OOM吧，尤其是在高并发的服务器端代码。最好的做法是采用Stream的方式边读取边存储(本地文件或database)。

正确的写法：

File pdf = toPdf(file);

另外，对于服务器端代码来说，为了系统的安全，至少需要对文件的大小进行限制。
