====== 学习 Python 的 12 个方式 ======
Created 星期四 19 九月 2019

学习 Python 的 12 个方式
2019-08-29 08:35

译自：https://opensource.com/article/19/8/dozen-ways-learn-python%E4%BD%9C%E8%80%85： Don Watkins
原创：LCTT https://linux.cn/article-11280-1.html%E8%AF%91%E8%80%85： Xingyu.Wang

这些资源将帮助你入门并熟练掌握 Python。



Python 是世界上最受欢迎的[1]编程语言之一，它受到了全世界各地的开发者和创客的欢迎。大多数 Linux 和 MacOS 计算机都预装了某个版本的 Python，现在甚至一些 Windows 计算机供应商也开始安装 Python 了。

也许你尚未学会它，想学习但又不知道在哪里入门。这里的 12 个资源将帮助你入门并熟练掌握 Python。

课程、书籍、文章和文档
1、Python 软件基金会[2]提供了出色的信息和文档，可帮助你迈上编码之旅。请务必查看 Python 入门指南[3]。它将帮助你得到最新版本的 Python，并提供有关编辑器和开发环境的有用提示。该组织还有可以来进一步指导你的优秀文档[4]。

2、我的 Python 旅程始于海龟模块[5]。我首先在 Bryson Payne 的《教你的孩子编码[6]》中找到了关于 Python 和海龟的内容。这本书是一个很好的资源，购买这本书可以让你看到几十个示例程序，这将激发你的编程好奇心。Payne 博士还在 Udemy[7] 上以相同的名称开设了一门便宜的课程。

3、Payne 博士的书激起了我的好奇心，我渴望了解更多。这时我发现了 Al Sweigart 的《用 Python 自动化无聊的东西[8]》。你可以购买这本书，也可以使用它的在线版本，它与印刷版完全相同且可根据知识共享许可免费获得和分享。Al 的这本书让我学习到了 Python 的基础知识、函数、列表、字典和如何操作字符串等等。这是一本很棒的书，我已经购买了许多本捐赠给了当地图书馆。Al 还提供 Udemy[9] 课程；使用他的网站上的优惠券代码，只需 10 美元即可参加。

4、Eric Matthes 撰写了《Python 速成[10]》，这是由 No Starch Press 出版的 Python 的逐步介绍（如同上面的两本书）。Matthes 还有一个很棒的伴侣网站[11]，其中包括了如何在你的计算机上设置 Python 以及一个用以简化学习曲线的速查表[12]。

5、Python for Everybody[13] 是另一个很棒的 Python 学习资源。该网站可以免费访问 Charles Severance[14] 的 Coursera 和 edX 认证课程的资料。该网站分为入门、课程和素材等部分，其中 17 个课程按从安装到数据可视化的主题进行分类组织。Severance（@drchuck on Twitter[15]），是密歇根大学信息学院的临床教授。

6、Seth Kenlon[16]，我们 Opensource.com 的 Python 大师，撰写了大量关于 Python 的文章。Seth 有很多很棒的文章，包括“用 JSON 保存和加载 Python 数据[17]”，“用 Python 学习面向对象编程[18]”，“在 Python 游戏中用 Pygame 放置平台[19]”，等等。

在设备上使用 Python
7、最近我对 Circuit Playground Express[20] 非常感兴趣，这是一个运行 CircuitPython[21] 的设备，CircuitPython 是为微控制器设计的 Python 编程语言的子集。我发现 Circuit Playground Express 和 CircuitPython 是向学生介绍 Python（以及一般编程）的好方法。它的制造商 Adafruit 有一个很好的系列教程[22]，可以让你快速掌握 CircuitPython。

8、BBC:Microbit[23] 是另一种入门 Python 的好方法。你可以学习如何使用 MicroPython[24] 对其进行编程，这是另一种用于编程微控制器的 Python 实现。

9、学习 Python 的文章如果没有提到树莓派[25]单板计算机那是不完整的。一旦你有了舒适[26]而强大的树莓派，你就可以在 Opensource.com 上找到成吨的[27]使用它的灵感，包括“7 个值得探索的树莓派项目[28]”，“在树莓派上复活 Amiga[29]”，和“如何使用树莓派作为 VPN 服务器[30]”。

10、许多学校为学生提供了 iOS 设备以支持他们的教育。在尝试帮助这些学校的老师和学生学习用 Python 编写代码时，我发现了 Trinket.io[31]。Trinket 允许你在浏览器中编写和执行 Python 3 代码。 Trinket 的 Python 入门[32]教程将向你展示如何在 iOS 设备上使用 Python。   

播客
11、我喜欢在开车的时候听播客，我在 Kelly Paredes 和 Sean Tibor 的 Teaching Python[33] 播客上找到了大量的信息。他们的内容很适合教育领域。

12、如果你正在寻找一些更通用的东西，我推荐 Michael Kennedy 的 Talk Python to Me[34] 播客。它提供了有关 Python 及相关技术的最佳信息。

你学习 Python 最喜欢的资源是什么？请在评论中分享。
