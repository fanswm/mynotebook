by [fmuellner](https://extensions.gnome.org/accounts/profile/fmuellner)
ONOFF
System extension
Substitute Alt-Tab with a window based switcher that does not group by application.

by [Maestroschan](https://extensions.gnome.org/accounts/profile/Maestroschan)

An easy way to manage "appfolders" (folders in the applications view) directly from the applications view. Bonus feature : it is possible to change the number of columns.

[Dash to Dock](https://extensions.gnome.org/extension/307/dash-to-dock/)
by [michele_g](https://extensions.gnome.org/accounts/profile/michele_g)
ONOFF
System extension
A dock for the Gnome Shell. This extension moves the dash out of the overview transforming it in a dock for an easier launching of applications and a faster switching between windows and desktops. Side and bottom placement options are available.
