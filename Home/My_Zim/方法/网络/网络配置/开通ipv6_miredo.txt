====== 开通ipv6 miredo ======
Created Monday 17 December 2018

vim [[/etc/modprobe.d/ipv6.conf]]
注释掉alias net-pf-10 off
开户alias ipv6 on
reboot

xubuntu:
sudo apt-get install miredo
sudo miredo


systemctl status miredo.service

测试：  http://test-ipv6.com/
