====== Git ======
Created Friday 24 November 2017

# 安装git
+ `sudo apt-get install git`

# 直接clone服务器:

+ `git clone git_user@[myip]:/home/git_user/repo_name.git`
> input passwd
+ `cd one`
+ `git checkout -b work_branch`
+ `git branch --set-upstream-to=origin/work work`
+ `git pull`
> input passwd

# 原始安装

## 密钥位置： 
+ `C:\users\administrator\.ssh\id_rsa`

## 在(home)win10中:
+ 打开git_shell
+ `ssh-keygen`

## 在(home)pi中:
+ `su root`
+ `cat /home/myusername/.ssh/id_rsa.pub >> /home/git/.ssh/authorized_keys`
+ `cat /mnt/usb/tmp/id_rsa.pub >> /home/git/.ssh/authorized_keys`
+ `su git`
+ `cd one.git`
+ `ssh-keygen`

+ `cd ~`
+ `mkdir one.git`
+ `chmod 760 one.git`
+ `cd one.git`
+ `git --bare init`

+ `sudo vim /etc/hosts`
> 在里面加入下面一行
```
mydomain.freedynamicdns.org gitserver
```

## 在win10中:

+ `mkdir onegit`
+ `git init`
+ `git remote add origin git@abpage.freedynamicdns.org:/home/git/one.git`
+ `vim test.txt`
+ `git add .`
+ `git commit -m"init git"`
+ `git fetch origin master`
+ `git branch --set-upstream-to=origin/master master`
> 解释:远程分支  本地分支
+ `gut pull`
+ `git push origin master`

+ `vim /etc/passwd`
> git禁用shell(git 禁用shell后,不能远程登录了)
+ `git checkout -b work`
> 新建分支work
+ `git push --set-upstream origin work`
+ `git push`
+ `git checkout -b work origin/work`
> 运行 git fetch，可以将远程分支信息获取到本地，再运行 git checkout -b local-branchname origin/remote_branchname  就可以将远程分支映射到本地命名为local-branchname  的一分支。 
> OK!

添加github.com远程仓库：
在浏览器打开同性交友网站github.com
新建一个仓库：例如test
在本地shell：
$ https://github.com/myusername/test.git
$ vim test.py
$ git add .
$ git commit -m"first commit"
$ git push
输入在github.com上的你的用户名：myusername
输入你的用户名的密码（myusername的）。
OK！

其他：
git config 删除一个变量：
$ git config --global --unset 项目
如：
$ git config --global --unset user.name
