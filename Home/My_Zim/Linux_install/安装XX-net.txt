====== 安装XX-net ======
Created Friday 24 November 2017

## 安装XX-net：
+ `git clone https://github.com/myusername/XX-Net`
+ `unzip ./XX-net.zip`
+ `sudo apt-get install libnss3-tools`
+ `sudo apt-get install python-openssl`
+ `cd XX-net`
+ `./start`

> linux提示：
> + python-openssl 已设置为手动安装。
> + libffi-dev 已设置为手动安装。

+ 打开 XX-net文件夹中的SwitchOmega，
+ 把Switchyomega.crx文件拖入扩展程序中，添加。

## 用FireFox安装XX-net：

+ 打开XX-net;
+ 同时已打开浏览器，下载以下链接：
> https://addons.mozilla.org/zh-CN/firefox/addon/pan/?src=search 
> 即：pan插件；

+ 添加过滤规则，添加GfwList，
+ 默认代理：GoAgent;
+ 记住点击pan插件，默认蓝色图标——表示“自动代理”，绿色——“全局代理”，灰色——“禁用代理”。
