将 pages/wxml/index.wxml 文件的内容做一些简单的修改：
<!--pages/wxml/index.wxml-->
<text>当前时间：</text>
 
 ​保存后工具刷新，模拟器并没有显示出当前的时间，这是因为我们并没有给 time 设置任何初始值，请打开 pages/wxml/index.js 文件，
 在 data 的大括号中加入：time: (new Date()).toString()。  
 
 WXML 通过 {{变量名}} 来绑定 WXML 文件和对应的 JavaScript 文件中的 data 对象属性。 
 
 展示格式：
<!--
{
time: (new Date()).toString()
}
-->
<text>当前时间：</text>
 
 属性值也可以动态的去改变，有所不同的是，属性值必须被包裹在双引号中，如下：
<!-- 正确的写法 -->
<text data-test=""> hello world</text>
 
 <!-- 错误的写法 -->
<text data-test=> hello world </text >
 
 没有被定义的变量的或者是被设置为 undefined 的变量不会被同步到 wxml 中
